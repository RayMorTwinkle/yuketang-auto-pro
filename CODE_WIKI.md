# 雨课堂助手（JS版）Code Wiki

## 1. 项目概述

雨课堂助手（JS版）是一个基于JavaScript开发的用户脚本，用于辅助雨课堂/荷塘雨课堂的学习过程。该项目提供了课堂习题提示、AI解答习题、自动作答、课件浏览等功能，旨在提升学习效率和体验。

- **项目版本**：1.21.1
- **适用平台**：Chrome、Edge浏览器
- **核心功能**：
  - 课堂习题提醒
  - AI辅助解答习题
  - 自动作答
  - 课件和幻灯片浏览
  - 自动进入课堂

## 3. 系统架构

雨课堂助手采用模块化架构，主要由以下几个核心部分组成：

1. **核心模块**：处理环境配置、存储管理和类型定义
2. **网络拦截模块**：拦截和处理雨课堂的API请求，获取课件和答题信息
3. **AI模块**：集成多种AI模型，用于解答习题
4. **UI模块**：提供用户交互界面，包括工具栏、设置面板等
5. **状态管理模块**：管理应用状态，包括题目状态、课件状态等
6. **截图模块**：用于获取幻灯片和题目的截图
7. **工具模块**：提供各种辅助功能，如答案格式化、提交等

## 4. 核心模块

### 4.1 主入口模块

主入口文件 [index.js](file:///workspace/ykt-helper/src/index.js) 负责初始化整个应用，包括加载样式、挂载UI、安装网络拦截器、启动自动作答轮询等。

### 4.2 网络拦截模块

网络拦截模块包括 [xhr-interceptor.js](file:///workspace/ykt-helper/src/net/xhr-interceptor.js) 和 [ws-interceptor.js](file:///workspace/ykt-helper/src/net/ws-interceptor.js)，用于拦截和处理雨课堂的API请求，获取课件和答题信息。

### 4.3 AI模块

AI模块集成了多种AI模型，包括Kimi、OpenAI、Gemini等，用于解答习题。主要文件包括 [kimi.js](file:///workspace/ykt-helper/src/ai/kimi.js)、[openai.js](file:///workspace/ykt-helper/src/ai/openai.js) 等。

### 4.4 UI模块

UI模块提供用户交互界面，包括工具栏、设置面板、AI面板等。主要文件包括 [toolbar.js](file:///workspace/ykt-helper/src/ui/toolbar.js)、[ui-api.js](file:///workspace/ykt-helper/src/ui/ui-api.js) 等。

### 4.5 状态管理模块

状态管理模块负责管理应用状态，包括题目状态、课件状态等。主要文件包括 [actions.js](file:///workspace/ykt-helper/src/state/actions.js) 和 [repo.js](file:///workspace/ykt-helper/src/state/repo.js)。

### 4.6 截图模块

截图模块用于获取幻灯片和题目的截图，为AI解答提供视觉信息。主要文件是 [screenshoot.js](file:///workspace/ykt-helper/src/capture/screenshoot.js)。

### 4.7 工具模块

工具模块提供各种辅助功能，如答案格式化、提交等。主要文件包括 [answer.js](file:///workspace/ykt-helper/src/tsm/answer.js) 和 [ai-format.js](file:///workspace/ykt-helper/src/tsm/ai-format.js)。

## 5. 关键类与函数

### 5.1 主入口函数

- **main()**：主入口函数，负责初始化整个应用
  - 加载样式和图标
  - 挂载UI
  - 安装网络拦截器
  - 启动自动作答轮询
  - 启动课件加载

### 5.2 网络拦截函数

- **installXHRInterceptor()**：安装XHR拦截器，用于拦截和处理雨课堂的API请求
- **installWSInterceptor()**：安装WebSocket拦截器，用于处理实时通信
- **getOnLesson()**：获取正在上课的课堂列表
- **checkinClass(lessonId, opts)**：进入课堂
- **getActivePresentationId(lessonId)**：获取当前活跃的课件ID

### 5.3 AI相关函数

- **queryKimi(question, aiCfg)**：调用Kimi文本模型
- **queryKimiVision(imageBase64, textPrompt, aiCfg)**：调用Kimi视觉模型
- **queryAI(question, aiCfg)**：通用AI查询函数
- **queryAIVision(imageBase64, textPrompt, aiCfg)**：通用AI视觉查询函数

### 5.4 自动作答相关函数

- **startAutoAnswerLoop()**：启动自动作答轮询
- **handleAutoAnswerInternal(problem)**：处理自动作答逻辑
- **actions.onUnlockProblem(data)**：处理题目解锁事件
- **actions.handleAutoAnswer(problem)**：处理自动作答请求

### 5.5 UI相关函数

- **installToolbar()**：安装工具栏
- **ui.showPresentationPanel(show)**：显示/隐藏课件面板
- **ui.showAIPanel(show)**：显示/隐藏AI面板
- **ui.toggleSettingsPanel()**：切换设置面板
- **ui.toggleTutorialPanel()**：切换教程面板

### 5.6 状态管理函数

- **repo.setPresentation(id, data)**：设置课件数据
- **repo.upsertSlide(slide)**：更新或插入幻灯片数据
- **repo.upsertProblem(problem)**：更新或插入题目数据
- **repo.pushEncounteredProblem(problem, slide, id)**：添加遇到的题目

## 7. 运行配置

### 7.1 安装方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 插件
2. 在 Tampermonkey 中创建新脚本，复制 [ykt-helper-1211.user.js](file:///workspace/release/ykt-helper-1211.user.js) 的内容
3. 保存并启用脚本

### 7.2 配置选项

在设置面板中可以配置以下选项：

- **习题提醒**：是否在新习题出现时显示通知
- **自动作答**：是否启用自动作答功能
- **自动作答延迟**：自动作答的延迟时间
- **AI API配置**：配置AI模型的API密钥
- **自动进入课堂**：是否自动进入正在上课的课堂

### 7.3 浏览器兼容性

- Chrome：145.0.7632.160及以下
- Edge：145.0.3800.82及以下

## 8. 功能说明

### 8.1 课堂提示

- 继承原版脚本的上课/下课/习题提示，防止上课摸鱼错过习题
- 可以查看这堂课的所有习题信息
- 可以在设置中修改习题提醒的方式

### 8.2 AI辅助学习

- 支持使用各种VLM智能解答课堂习题，需要在设置界面自行设置密钥
- 支持所有题型，包括单选，多选，填空，简答，投票
- Agent模式：根据提问的难度、预留作答时间、题目类型等调用合适的工作流以提高答题正确率
- OCR功能：可以配置OCR模型（或使用VLM）便捷识别、复制PPT里的文字
- 翻译功能：可以便捷翻译OCR结果

### 8.3 AI自动作答

- 可以在设置界面设置是否自动作答，以及自动作答的延迟时间
- 自动作答支持所有题型，包括单选，多选，填空，简答，投票
- 如果未打开问题列表，会默认解答最新的题目
- 如果已打开问题列表并选择了某个问题，会解答选中的问题
- 在没有配置Apikey而且全程挂机的情况下，本插件将随机作答

### 8.4 查看课件和习题列表

- 可以查看所有幻灯片，默认只显示习题页面
- 可以显示全部幻灯片
- 可以导出为PDF
- 可以查看需要提交或者补交的习题
- 支持对不懂的幻灯片进行提问（需要接入AI）

### 8.5 自动进入课堂

- 登录后进入雨课堂主页，确认已经在雨课堂主页运行脚本
- 在设置中打开“自动进入课堂”，刷新页面
- 在开始上课后会自动进入正在上课的页面

### 8.6 补交习题

- 可以通过「查看课件和幻灯片」- 题目列表 - 强制补交 来在**下课前**修改或补交习题
- 补交习题需要自己写json格式

## 9. 开发与构建

### 9.1 开发环境设置

1. 克隆项目到本地
2. 进入 `ykt-helper` 目录
3. 运行 `npm install` 安装依赖
4. 运行 `npm run dev` 启动开发模式

### 9.2 构建流程

1. 进入 `ykt-helper` 目录
2. 运行 `npm run build` 构建项目
3. 构建产物会生成在 `dist` 目录中

### 9.3 代码规范

- 使用ES6+语法
- 采用模块化结构
- 代码注释清晰
- 遵循JavaScript最佳实践

## 10. 部署与使用

### 10.1 部署方法

1. 构建项目，生成用户脚本文件
2. 将生成的用户脚本文件复制到 `release` 目录
3. 在 [Greasy Fork](https://greasyfork.org/) 上发布更新

### 10.2 使用方法

1. 在浏览器中安装 Tampermonkey 插件
2. 安装雨课堂助手用户脚本
3. 登录雨课堂账号
4. 进入课堂页面，即可看到工具栏
5. 根据需要使用各种功能

### 10.3 注意事项

- 本工具仅供学习参考，建议独立思考解决问题
- AI解答功能需要使用VLM API，可能会产生API使用费用
- 本项目不会线上存储用户的任何个人信息，只会将相关的题目信息上传给VLM/LLM
- 自动作答可能有误，可能作答失败。本插件不对可能造成的成绩影响负责
- 暂时不支持扫描雨课堂**动态二维码**
